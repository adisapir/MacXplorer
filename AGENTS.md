# Repository Instructions

## Git

- Display every Git command in chat before running it.
- Suggest missing Git steps that follow best practices, calibrated for a single-contributor repository.
- Before merging a branch into `main`, increment the app version by `0.01` unless the developer specifies otherwise, commit that version change as the final branch commit, and then merge.
- Before merging a branch into `main`, ask the developer whether they want to update `CHANGELOG.md`. Do not merge until they answer.
- Don't add "Co-Authored-By: " signatures
